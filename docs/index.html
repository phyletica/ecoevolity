About¶

Ecoevolity is a software package for Bayesian inference of shared evolutionary events developed by the Phyletica Lab. The software comprises two primary sets of tools:

The ecoevolity tools (ecoevolity, simcoevolity, sumcoevolity, DPprobs) are for comparing divergence times among pairs of populations or species. Click here for more details.

The phycoeval tools (phycoeval, simphycoeval, sumphycoeval) provide a fully phylogenetic approach to jointly inferring a species tree, shared divergence times, and multifurcating divergences. Click here for more details.

Pycoevolity¶

Pycoevolity is a companion Python package for preparing input data and summarizing the output of ecoevolity.
